# CKA Preparation

This folder contains a structured study plan for Kubernetes Certified Kubernetes Administrator (CKA) preparation.

Files:
- `study_plan.md` — weekly study plan and setup steps.
- `topics.md` — CKA domains and key topics to practice.
- `practice_tasks.md` — hands-on tasks to reinforce learning.
- `resources.md` — recommended books, courses, and tools.
