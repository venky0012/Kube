# CKA Resources

## Official Documentation
- Kubernetes docs: https://kubernetes.io/docs/
- CKA exam curriculum: https://www.cncf.io/certification/cka/

## Training
- Official Kubernetes training courses.
- Free interactive labs from Katacoda / KubeAcademy.
- YouTube tutorials and practice sessions.

## Practice
- Kubernetes Hands-on: use Minikube, KIND, or a cloud lab.
- CKA practice exam simulators.
- GitHub repositories with Kubernetes challenge tasks.

## Tools
- `kubectl` for cluster management.
- `kubeadm` and `minikube` for local cluster setup.
- `kubectx` / `kubens` for context and namespace switching.
- `k9s` for terminal-based cluster navigation.
